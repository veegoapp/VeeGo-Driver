Tasks:

1) Remove sensitive data from production logs

Check and fix these areas:
- app/login.tsx
- app/verify-otp.tsx
- lib/api.ts

Requirements:
- Remove or protect console logs that expose:

2) Fix location tracking data loss

File:
- hooks/useActiveLocationTracking.ts

Problem:
Pending locations are currently removed before confirming successful upload.

Requirements:
- Do not delete stored locations before the server confirms successful upload.
- Only remove pending locations after sendBatch succeeds.
- If upload fails:
  - keep the stored locations
  - allow retry later.
- Do not change the backend request format.

- lib/auth.ts

Requirements:

- Do not change token handling except adding the tracking cleanup.
