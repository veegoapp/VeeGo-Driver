## 3. API INTEGRATION

**Base URL:** `process.env.EXPO_PUBLIC_API_URL` — defaults to `https://` prefix if not http-prefixed. Written to `.env` by `setup.sh` from the `BACKEND_URL` secret.

**Global error handling in `lib/api.ts`:**
- 15-second request timeout via `AbortController`
- `401` → silent token refresh via `/auth/refresh`; on failure, clears tokens
- `403 account_suspended` → triggers global `_onAccountSuspended` callback → `/suspended` screen
- Network/abort errors → `ApiError(status: 0)`
- Non-OK responses → parsed JSON/text body thrown as `ApiError`
